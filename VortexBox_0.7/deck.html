Wiki


Some apps on the deck need to be installed on your VortexBox before you can use them. If you would like to enable these wonderful features, here are some tips to help you get started.

 

Shellinabox is a great tool. It allows a command-line interface to VortexBox from your web browser. Learn how to install shellinabox on your VortexBox here.

 

Webmin allows you to manage and monitor the linux operating system from a GUI. Visit the webmin wiki here for instructions on installation.

 

Subsonic can connect to your VortexBox inside and outside your network with very little user configuration. Upgrade here.

 

Transmission is a great little bittorrent client you can install on your vortexbox. Install here or click "Upgrade" on the deck.

 

Bliss allows you to conveniently manage your album cover art. Install here or click "Upgrade" on the deck.

 

MusicIP is similar to iTunes Genius. Install by running setup-musicip from command line.

Options


Options

      To change the stored IP address for your VortexBox, or set your subsonic domain, visit the options menu. You can also adjust by clicking the wrench on the VortexBox icon in the app menu. For more details, visit the wiki for this app.

 

About

      VortexBox Web App: version 0.6

 

Uninstall

      In the Chrome app menu, click the wrench on the VortexBox icon and select "Uninstall".

 

Source

      To find out where the source code is located on your machine, visit the chromium website.
